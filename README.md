## Local Installation

  [ clone the repo ]

  ```bash
  git clone https://github.com/Moriarity-2k/SnakeAndLadder/
  ```

  [ navigate to the folder in terminal ]
  ```bash
  cd SnakeAndLadder
  ```

  [ install the dependencies ]
  ```bash
  npm install
  ```

  [ Run the project Locally ] 
  ```bash
  npm run dev
  ```
